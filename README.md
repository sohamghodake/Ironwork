# Ironwork

Distributed job scheduling and execution platform, built to demonstrate real
distributed-systems patterns end to end: Raft consensus, backpressure, CRDT
eventual consistency, transactional outbox, and full observability.

> **Status: Phase 0** — cluster scaffolding. The compose cluster boots, proto
> contracts compile, migrations apply, and one health check passes end to end.
> No Raft, no CRDT yet — see the phase plan below.

Full quickstart and architecture docs land alongside the Phase 0 deploy files.
